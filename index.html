選秀辦法
  1. 選秀時間
    10/15(六)：09:00~22:00
    10/16(日)：20:00~22:00
  2. 據參賽人數取12~20名玩家作為隊長，挑選方式以 二代 X power 為準，由高至低排序，如遇同樣 Rank 區間玩家則以亂數決定選秀順序，如遇同樣 Rank 區間玩家大於隊長需求數量，一樣會以亂數抽選決定隊長資格
  3. 選秀第一日，第一輪選秀將由各隊長填寫選秀志願，將依隊長順位最低的隊伍優先排入選手
  4. 選秀第二日公布第一輪選秀結果，第二輪選秀將由順位最高的隊長開始挑選(1~20) 最後再由最低順位隊長挑選(20~1)完成選秀流程
  5. 如隊長選秀日未能如期參與（失聯15分鐘以上），將會取消隊長資格，由次高順位的玩家遞補
對戰規則

    為確保比賽進程順暢，請遵守以下規範：

  1. 初賽及決賽對戰地圖全地圖隨機抽選
    初賽及決賽對戰場地皆以不重複為原則，請房主單場對戰結束後進行ban圖。
  2. 若無特殊需求，請由隊伍數字代號較小的一方負責開房。
  3. 每場對戰皆須在「所有隊伍」確認好前一階段對戰結果後再行開始，主辦會發布可開始進行對戰的訊息。
  4. 請勿對選手或隊伍進行挑釁、誹謗中傷等行為。
  5. 請保持充足睡眠規律生活不要睡過頭......(*´-ω-`人)

問題回報

    Q.規則／場地／隊伍設定錯了，怎麼辦？

  1. 若有選擇錯誤情形，請重開房，雙方以同樣地圖規則、同樣武器組成進行對戰。
  2. 若房主方連續兩次選擇錯誤，則該場對戰結果判定為對方勝利。
  3. 若有選擇錯誤情形發生，請非房主方不要推動規則，由房主方將該場對戰迅速結束。
  4. 若非房主方推動規則，則該場對戰繼續進行，對戰結果列入計分。

    Q.斷線怎麼辦？

  1. 對戰開始60秒內斷線：請重開房，雙方以同樣地圖規則、同樣武器組成、裝備進行對戰。
  2. 對戰開始60秒內同一隊伍參賽者第二次斷線：請繼續進行對戰。
  3. 對戰開始60秒後斷線：請繼續進行對戰。
  4. 若為開房方斷線，且為必須重新開房的情況，請由另一方重新開房。
    ＊ 60秒內：倒數時間顯示4:00(含)以前，如有疑慮請務必錄影。

    Q.衝線怎麼辦？

  1. 比賽開始前請務必提前確認個人網路狀況，如開房後有衝線情形請立刻於Discord內之「問題回報」頻道提出，由主辦方協助調整開房事宜。
